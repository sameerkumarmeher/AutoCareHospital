# AutoCare Hospital Analytics

## 📊 Project Overview
This repository contains SQL queries, scripts, and analysis for AutoCare Hospital’s discharge data.  
The goal is to provide actionable insights to stakeholders about patient discharges, hospital efficiency, and demographic trends.

---

## 🏥 Business Questions
The stakeholders would like to know:

1. **Total Discharges**  
   - Count of all patient discharges in the dataset.

2. **Average Daily Discharge Rate**  
   - Average number of discharges per day.

3. **Average Length of Stay (ALOS)**  
   - Average number of days patients spend in the hospital before discharge.

4. **Distribution of Discharges by Age Group**  
   - Categorize patients into age groups (e.g., 0–18, 19–35, 36–50, 51–65, 65+) and show discharge counts.

5. **Distribution of Discharges by Gender**  
   - Male vs. Female discharge counts.

6. **Distribution of Discharges by Day of the Week**  
   - Analyze which weekdays have higher or lower discharge volumes.

---

## ⚙️ Methodology
- **Data Source**: Hospital discharge records (structured in SQL tables).  
- **Tools Used**: SQL Server / MySQL, Power BI (optional for visualization).  
- **Approach**:
  - Write SQL queries to calculate totals, averages, and distributions.
  - Use `GROUP BY` for age groups, gender, and weekday analysis.
  - Create views for reusable queries.
